# Privacy Policy

## Obsi Redbook Clipper

Obsi Redbook Clipper runs locally in the user's browser.

### What the extension accesses

- The currently open Xiaohongshu post page or public WeChat article page
- User-configured extension settings stored in Chrome sync storage
- Downloaded Markdown or JSON files only when the user explicitly requests a download

### What the extension does not do

- It does not send captured page content to a remote server
- It does not collect analytics
- It does not track browsing activity outside supported pages
- It does not upload cookies, account data, or personal vault content

### Local storage

The extension stores only small configuration values such as:

- Obsidian vault name
- Obsidian folder path

These values are stored using Chrome extension storage.

### User control

Users control when content is extracted, copied, downloaded, or sent to Obsidian. No background syncing or automatic upload is performed.
